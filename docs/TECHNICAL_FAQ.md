# Technical FAQ

## Q: How do I contribute code?
A: Fork repo, pick issue from PROJECTS_NEW.md, submit PR.

## Q: What's the tech stack?
A: Coreum (CosmWasm) for logic, XRPL for settlement, simple HTML/JS frontend.

## Q: How do developers get paid?
A: Via grants from DAO treasury after community vote (see CONTRIBUTOR_GUIDE.md).

## Q: Can I ask technical questions?
A: Yes, open a GitHub Issue with the 'technical' label.

## Q: Is there real-time chat?
A: No, we use GitHub Issues for all discussions to maintain permanent, searchable documentation.

## Q: How quickly will you respond to PRs?
A: All technical contributions reviewed within 48 hours.

## Q: What if I'm not good at explaining things verbally?
A: Perfect. We value code and written documentation over verbal communication.

## Q: How are technical decisions made?
A: Through GitHub Issues discussions, followed by formal proposals when consensus emerges.
