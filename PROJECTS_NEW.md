# PhoenixPME - Precious Metals on TX Blockchain

A decentralized auction platform for physical gold, silver, platinum, and palladium built on the TX blockchain (Coreum + Sologenic merger).

## Built With
- TX Blockchain / Coreum / Sologenic
- CosmWasm Smart Contracts
- Next.js / React
- Rust
- Keplr Wallet Integration
- Leap Wallet Integration
